About Mappia

CSR Maps makes your interactive maps easy to publish online

   The service is free of charge, and fully customizable with low effort. To publish your data we offer two options, one option does not require a private server but only an account in an online file storage service such as GitHub, BitBucket and many others, the other requires a private server configured with a Web Map Service (WMS). The platform offers a set of customizable ready-to-use tools called elements. These elements allows many different interactions such as inspect values, apply map algebra, create interactive charts and many others. This platform can integrate multiple data sources simultaneously, such as CSR Database, other online databases, and user maps.

   Create your layout in the editor mode, an interactive and easy-to-use interface that allows the full control of the platform itself. You have the full control of the customizations by: choosing maps, using map interactive elements, getting user inputs, creating interactive charts, defining a map algebra expression, alot of others customizations and creating your own tools. These maps can be easily embedded into sites, or shared by the project url.

   This is a complete solution for online mapping designed for all users, the ones that just want to create an online map viewer, to the ones that want to simulate sceneries using your data online. So for all users, we fully documented our elements with examples, and a tutorial showing the simplest way to publishing your interactive maps online and create a shareable platform. All CSR Platforms were made within Mappia as a fully accessible and editable portfolio. No install is required, try it online now.
